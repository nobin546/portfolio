# Deployment Guide

## Static deployment options

### GitHub Pages
1. Create a GitHub repository and push the project files.
2. In repository settings, enable GitHub Pages from the root branch.
3. The site will be published at your GitHub Pages URL.

### Netlify
1. Drag and drop the project folder into Netlify.
2. Set the publish directory to the project root.
3. Deploy.

### Vercel
1. Import the repository into Vercel.
2. Set the framework to static or use the root directory.
3. Deploy.

## Notes
- The site uses only static HTML, CSS, and JavaScript.
- No build step is required for deployment.
