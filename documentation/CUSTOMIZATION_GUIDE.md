# Customization Guide

## Replace personal details
Update the following values in index.html:
- Name and headline text
- Email, WhatsApp, GitHub, LinkedIn, and Facebook links
- Project titles and descriptions
- CV download link

## Replace images
Place new files inside the public/ directory and update the src attributes in index.html.

## Adjust colors
Edit the CSS variables in styles.css to change accents, surfaces, and background tones.
